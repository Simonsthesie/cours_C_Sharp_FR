# Cours C# — Apprentissage Guidé

Cours de C# en français avec 10 modules et des exercices pratiques.

## Modules

| Module | Thème |
|--------|-------|
| 01 | Les bases : variables et types |
| 02 | Opérateurs et expressions |
| 03 | Structures de contrôle (if, switch, boucles) |
| 04 | Tableaux et collections |
| 05 | Méthodes et paramètres |
| 06 | Introduction à la POO |
| 07 | Héritage et polymorphisme |
| 08 | Interfaces et abstraction |
| 09 | Gestion des exceptions |
| 10 | Projet final : gestionnaire de tâches |

## Structure

- **cours/** — Pages HTML pour la théorie (ouvrir `index.html`)
- **01-Les-Bases/**, **02-Operateurs/**, etc. — Modules avec exercices

## Prérequis

- .NET SDK (`dotnet --version`)
- Un éditeur de code

## Exécuter un exercice

```bash
cd "01-Les-Bases\Ex01-Variables"
dotnet run
```
