Introduction au C#

Qu'est-ce que le C# ?

C# (prononcé « C sharp ») est un langage de programmation moderne, créé par Microsoft en 2000. Il fait partie de la famille des langages dérivés du C, comme C++ ou Java, mais a été conçu pour être plus simple et plus sûr. Aujourd'hui, C# est open source et fonctionne sur Windows, Linux, macOS, et même dans le navigateur via WebAssembly.

Pourquoi apprendre le C# ?

Arguments en faveur du C#

  • Polyvalent : applications desktop, web, mobiles, jeux vidéo, cloud…
  • Syntaxe claire : lisible et proche du langage naturel
  • Écosystème solide : .NET est mature, bien documenté et très utilisé en entreprise
  • Demande sur le marché : nombreuses offres d’emploi en France et dans le monde
  • Gratuit et open source : .NET peut être utilisé sans licence payante

Utilisation au quotidien

Le C# sert dans de nombreux contextes professionnels :

DomaineExemples
Applications webASP.NET Core — sites, API REST, backends
Applications desktopWPF, WinForms, applications métier Windows
Jeux vidéoUnity utilise C# pour le développement de jeux
MobileApps iOS et Android avec .NET MAUI
Cloud & servicesAzure, microservices, APIs
Outils métierScripts d’automatisation, logiciels internes

Spécificités du langage

  • Typé fortement : chaque variable a un type défini, ce qui limite les erreurs
  • Orienté objet : classes, héritage, polymorphisme sont au cœur du langage
  • Gestion automatique de la mémoire : pas besoin de gérer manuellement l’allocation/libération (contrairement au C/C++)
  • Langage moderne : évolutions régulières (async/await, LINQ, pattern matching…)
  • Interopérable : s’intègre bien avec d’autres technologies Microsoft et open source

C# vs autres langages

Si tu connais déjà Java, la syntaxe te sera familière. C# reprend beaucoup d’idées de Java en les simplifiant et en y ajoutant des fonctionnalités modernes. Comparé à Python, C# est plus verbeux mais plus performant et typé. Par rapport au JavaScript/TypeScript, C# apporte un typage plus strict et un modèle objet plus rigoureux.

Comment faire les exercices ?

Tu peux choisir la méthode qui te convient le mieux :

1. Directement dans le navigateur

Chaque module propose des exercices avec un éditeur intégré. Tu écris ton code, tu cliques sur ▶ Lancer et tu vois le résultat immédiatement. Idéal si tu n'as pas encore installé .NET ou si tu veux tester rapidement.

2. Dans ton IDE (VS Code, Visual Studio…)

Tu préfères travailler en local ? Les exercices sont dans les dossiers Ex01, Ex02, etc. Ouvre le fichier Program.cs, écris ton code, puis exécute avec :

cd "01-Les-Bases\Ex01-Variables"
dotnet run

Pour un autre exercice, adapte le chemin, par exemple :

cd "01-Les-Bases\Ex02-CalculSimple"
dotnet run

Prérequis : .NET SDK installé (dotnet --version pour vérifier).
